Combat value: Combat value is the sum of the value of all statistics.

It helps in measuring which statistics are more valuable than others.

In a well-balanced combat system, characters with similar combat value should have roughly the same chances of winning against each other, unless a counter-play to their strategy is applied.
{{ system.hero.total_value }}
Attributes Attributes describe the effectiveness of character in combat.

You can balance their properties in section "Ruleset".

Some attributes can be turned off by setting their value to 0.
Priority Priority describes the frequency with which the character will use either a physical or magical attack based on the system of chance.

To completely nullify the chance of either attack happening, you can set its value to zero.

Sum of both values must be equal to 100%, otherwise it will be set to a default value.
